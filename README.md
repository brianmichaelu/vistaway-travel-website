# VistaWay Travel & Tours

A premium Next.js travel agency demo website built with TypeScript, App Router, Tailwind CSS, and a component-based structure.

## Run locally

```bash
npm.cmd install
npm.cmd run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm.cmd run build
```

## Images

All images are local `.png` files stored in:

```text
public/images/
```

Replace the placeholder images with real client images later, keeping the same file names or updating the paths in the components.
